# Orbit Note - Two Costumes, One Clock

Date: 2026-07-14
Decision: Promote the kernel, use Session Deck as control, keep Mutation as the
R&D challenger, and stop implementation before product commitment.

Both disposable interfaces completed the same headless and real-browser
scenario with real Tone.js voices. The Session Deck is visually easier to enter;
the Mutation Instrument exposes more reversible depth. That is a meaningful
interaction difference, but it is not yet a musical preference.

The loop therefore stops at the intended boundary. Q2-L is a separate human
listening session. No scene, slot, recording, macro, capture, recovery, or agent
concept moves into the durable song architecture before that result.
