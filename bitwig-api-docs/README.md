# Bitwig API Documentation

This directory intentionally does not vendor Bitwig Controller API documentation.

Earlier local notes in this repository were generated with LLM assistance from
Bitwig's official documentation and may have included close paraphrases or
copied reference material. They were removed before public release because this
repository does not have explicit permission from Bitwig GmbH to redistribute
that documentation.

Use the official Bitwig Studio documentation and controller-script resources
provided by Bitwig for API reference material.

If you generate private local notes from the official documentation, keep them
untracked and do not commit copied or closely paraphrased Bitwig documentation
to this repository.

Git does not track empty directories, so this file is kept as the public
placeholder for the removed reference corpus.
